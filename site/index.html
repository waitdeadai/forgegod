WHAT MAKES IT DIFFERENT

Multi-Model Auto-Routing

Route tasks to the right model automatically. Local Ollama for bulk coding ($0), cloud for review, frontier for hard problems. Circuit breaker + fallback chains. Thompson sampling learns which model works best for what.

OpenAI + Anthropic + Ollama + OpenRouter

Ralph Loop

24/7 autonomous coding from a PRD. Progress lives in git, not LLM context. Fresh agent per story. No context rot, ever.

Reflexion Coder

3-attempt code gen with escalating models. Local free, then cloud cheap, then frontier. AST validation at every step.

5-Tier Memory System

Episodic (what happened) + Semantic (what I know) + Procedural (how I do things) + Graph (how things connect) + Error-Solutions (what fixes what). Memories decay, consolidate, and reinforce. The most advanced memory of any open-source coding agent.

Cross-session learning
