## Overview

MiniMax 2.7 High Speed is the **operational default model** for ForgeGod
coding-agent work as of 2026-04-12. This document captures the known
configuration and integration approach.

## What Is Known

### Provider Configuration
- MiniMax is added as the 10th route surface in `forgegod/router.py`
- Provider family: `minimax`
- Model identifier: `minimax:2.7-highspeed` (or equivalent 2.7 variant)
- Integration: OpenAI-compatible API surface (standard ForgeGod provider path)

### Integration Approach
ForgeGod uses OpenAI-compatible routing for all providers. Adding MiniMax:
1. Add `minimax` to the provider families list
2. Configure base URL + API key in `.forgegod/config.toml` or via env var
3. Add to `preferred_provider` or use as explicit model override

### Why MiniMax 2.7 as Default
- User direction: MiniMax 2.7 High Speed is the specified operational default
- ForgeGod remains provider-agnostic — the default is a working convention
- When web search is available, verify MiniMax benchmarks against the current
  `glm-5.1` and `openai-codex:gpt-5.4` baselines
